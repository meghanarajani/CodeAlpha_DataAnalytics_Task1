# CodeAlpha Data Analytics Internship - Task 1

## 📌 Project Overview
Successfully scraped 60 books data from [books.toscrape.com](https://books.toscrape.com) using Python for the CodeAlpha Data Analytics Internship.

## 🛠️ Tech Stack
- **Python 3.x**
- **BeautifulSoup4** - For HTML parsing
- **Pandas** - For data manipulation & CSV export
- **Requests** - For HTTP requests

## 📊 Data Extracted
The script extracts the following information for each book:
1. **Title** - Book name
2. **Price** - Price in £
3. **Rating** - Star rating (One to Five)
